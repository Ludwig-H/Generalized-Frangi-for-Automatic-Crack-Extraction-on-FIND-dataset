# Protocole CrackSAM publié

`lists/` contient les listes originales utilisées pour rapprocher les
évaluations locales du protocole CrackSAM. Elles sont centralisées ici pour que
le code actif et la référence SAM 1 utilisent une seule copie.

Ces listes constituent de la donnée expérimentale versionnée : ne pas les
modifier silencieusement. La phase 0 de la nouvelle feuille de route ajoutera un
manifeste de SHA-256 et des groupes d'images physiques avant de définir le
holdout confirmatoire.
