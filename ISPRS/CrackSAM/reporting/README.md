# Génération de rapports

Les scripts de ce dossier lisent des sorties expérimentales et produisent des
rapports ou galeries. Ils ne lancent pas d'entraînement et ne modifient pas les
checkpoints.

- `generate_frangi_milestone_report.py` : rapport des jalons ;
- `generate_frangi_preprocessing_gallery.py` : galerie de prétraitement ;
- `analyze_frangi_chrominance_cpu.py` : sonde chrominance/polarité sur CPU.
